Under the Hood

How It Works

From training to real-time detection in four simple steps.

01

Train the Model

Models were trained on Roboflow using annotated datasets with RF-DETR (Small) architecture for efficient, accurate detection.

02

Upload Image / Camera

You provide an image or live camera frame. The image is encoded and sent securely to the Roboflow inference API.

03

AI Inference

Roboflow's hosted cloud runs the RF-DETR model on your image and returns predictions including bounding boxes and confidence scores.

04

Visualize Results

Bounding boxes are drawn directly on your image using HTML5 Canvas, with class labels and confidence scores displayed in real time.

Integration

Simple to Integrate

A single API call is all it takes to run inference against your trained model.

JavaScript — Roboflow inferencejs 
const { InferenceEngine } = inferencejs;
const inferEngine = new InferenceEngine();

// Load the model worker once — downloads & caches on-device
const workerId = await inferEngine.startWorker(
  "face_liveliness_detection_v2",
  1,
  "YOUR_PUBLISHABLE_KEY"
);

// Pass any HTMLImageElement or HTMLCanvasElement
const predictions = await inferEngine.infer(workerId, imgElement);
// predictions → [{ class, confidence, x, y, width, height }]
